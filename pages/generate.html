          Notes

          Quelques explications...


          ΣComp: ((Valeurs de base + Pts de compétences sup. + Archétype) - (Magie - Pts décrémentés sur le COM)) + Magie = Total des pts de compétences (doit être le même résultat)

          Nom:
          • Logique: tire un nom au sort de façon cohérente avec le peuple sélectionné

          Peuple :
          • Logique: tire un peuple au sort sur la table des origines (Livre I, p29)

          Groupe :
          • Logique: tire un groupe de carrière au sort sur la table des carrières (Livre I, p55-56) en fontion du peuple sélectionné

          Compétences (définit le mode de calcul des points venant s'ajouter aux valeurs de base des compétences) :
          • Répartition : répartit 120 points dans les compétences (sauf magie) par tranches de 5
          • Tirage : assigne la somme de 2d10 dans chaque compétence (sauf magie)
          • Tirage (tranche de 5) : assigne 0, 5, 10, 15 ou 20 dans chaque compétence (saug magie)

          Mode:
          • Normal: permet de créer un personnage-joueur ou un premier rôle selon les règles normales de création
          • Spécialiste: le personnage dispose de caractéristiques supplémentaires (voir détails d'attribution des caractéristiques)
          • Champion: idem
          • Héros mineur: idem
          • Héros majeur: idem
            Note: la génération de l'équipement reste toujours calquée sur le mode normal. Il peut donc être intéressant de l'adapter pour les autres modes.
